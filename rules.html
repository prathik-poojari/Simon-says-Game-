Description

Simon Says is a classic memory game where players must remember and repeat a sequence of colors. This project is a web-based implementation of the Simon Says game using HTML, CSS, and JavaScript.

How to Play

Start the game by clicking the "Start" button. Simon will play a sequence of colors. Remember the sequence. Click or tap the colors in the same order as Simon played them. If you make a mistake, the game will end, and your score & High score will be displayed.
